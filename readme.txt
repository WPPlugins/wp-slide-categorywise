Wp Slide Categorywise is a simple WordPress plugin that creates a beautiful responsive categorywise slider for your website or blog.

== Description ==

> This plugin helps you to create responsive categorywise slider for each categories in a tab display.
> You just need to create the categories and then add slides for each categories and rest of work is done by the this plugin.
> This plugin contains a settings page so you get a customized slider according to your requirement.
> Wp slide categorywise is very easy to customize and manage with various options.
> It shows the total number of slides in each of the categories.
> It allows to edit and delete slides in any category section.
> Image title/description will be shown as a caption on slide image.

Wp Slide Categorywise is a simple WordPress plugin that generates a responsive category wise slider on your website or blog.simply insert [wpsc_categorywise_slides] in any page or post content and the form will appear or paste this php code `<?php echo do_shortcode( '[wpsc_categorywise_slides]' ); ?>` in any template.

This plugin supports only English(default) both in the back-end and  front-end of the site.

Features:

* Easy to install.
* It is responsive slider.
* Easy to understand coding standard.
* Easy to customize slider settings.
* Easy to add images in slides section.
* Easily compatible with any wordpress theme.

== Installation ==

1. Upload the `category-slider` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin using the 'Plugins' menu in your WordPress admin panel.
3. You can adjust the necessary settings using your WordPress admin panel in "Wp Slide Categorywise".
4. Create a page or a post and insert the shortcode [wpsc_categorywise_slides].
5. You can add `<?php echo do_shortcode( '[wpsc_categorywise_slides]' ); ?>` shortcode in any template.
6. To start adding slides you need to add the categories first from post categories section in wp-admin.
7. Maximum image upload size is 2MB.
8. After you can start adding slides from Wp Slide Categorywise tab.
